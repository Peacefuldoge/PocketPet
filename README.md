# 企鹅女孩桌宠 2.6

原生 Android 桌面宠物，使用透明 `WindowManager` 悬浮窗实现。

## 功能

- 真正透明的悬浮窗背景
- 向右使用 25 张完整角色整帧绘制的行走循环
- 每帧的身体、双脚、双臂/翅膀和尾巴均在同一张图中完整绘制
- 不使用身体底稿、独立脚层或独立尾巴层进行拼接
- 向左动画仅由完整向右帧整体水平镜像生成
- 头部、头发、躯干重心、双脚与尾巴随步态连续变化
- 拖到手机左/右边框并松手后，会播放钻出屏幕的停靠动画并只露出朝向屏幕内侧的半个头
- 点击露出的半个头会反向播放动画，完整回到屏幕内
- 长按宠物显示“投喂”按钮，食物飞向宠物后复用开心跳跃动画
- 约 60Hz 坐标更新，随机选择全屏二维目标点巡游
- 站立、行走、开心、趴下、呼吸睡眠和起身状态自动切换
- 连续移动三段后长时间静止会趴下休息，点击或拖动可唤醒
- 点击角色触发开心姿态，拖动可改变位置
- 前台服务保持运行，可从通知或控制页面停止

## 构建

推荐使用 Android Studio、JDK 17、Android SDK 34：

```bash
./gradlew assembleDebug
```

Windows PowerShell：

```powershell
.\gradlew.bat assembleDebug
```

APK 输出：`app/build/outputs/apk/debug/app-debug.apk`。
